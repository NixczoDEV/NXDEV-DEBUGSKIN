1. Drag NXDEV-DEBUGSKIN in your resource folder
2. Setup clogs.lua
3. Add ensure NXDEV-DEBUGSKIN in your server.cfg
4. Add add_ace group.admin "command.debug" in your server.cfg
5. Start the server
6. Use this IG : /debug [ID]
